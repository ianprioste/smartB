# SmartBling

SmartBling é uma aplicação profissional de gerenciamento de produtos para a plataforma Bling, desenvolvida em Python (FastAPI) e React.

## Quick Start

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
python main.py configurar
python main.py run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
