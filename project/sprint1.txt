⚡ QUICK START (5 minutos)
═══════════════════════════════════════════════════════════════════════════════

1. Dentro da pasta backend:
   $ cd backend
   $ cp .env.example .env

2. Editar .env com credenciais Bling:
   BLING_CLIENT_ID=seu_id
   BLING_CLIENT_SECRET=seu_secret

3. Iniciar serviços:
   $ docker-compose up -d

4. Setup do projeto:
   $ ./setup.sh    (Linux/Mac)
   $ setup.bat     (Windows)

5. Rodar servidor:
   $ python run.py

6. Rodar worker (outro terminal):
   $ celery -A app.workers.celery_app worker --loglevel=info
   (Windows) $ python celery_worker_windows.py

✓ Pronto! API em http://localhost:8000/docs
