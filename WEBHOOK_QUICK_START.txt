---

## ⚡ Registrar Todos de Uma Vez

```powershell
# Via script
.\register-bling-webhook.ps1 -Command register

# Via curl
curl -X POST http://localhost:8000/webhooks/bling/register-all-webhooks
```

## 📋 Tabela de Cópia/Cola

```
Pedidos     | http://localhost:8000/webhooks/bling/orders     | SmartBling Order Sync
Produtos    | http://localhost:8000/webhooks/bling/products   | SmartBling Product Sync
Estoque     | http://localhost:8000/webhooks/bling/stock      | SmartBling Stock Sync
Fornecedores| http://localhost:8000/webhooks/bling/suppliers  | SmartBling Supplier Sync
```

---

## ✅ Como Funciona

```
1. Dados mudam no Bling
   ↓
2. Bling envia POST para seu webhook
   ↓
3. App sincroniza automaticamente
   ↓
4. Dados atualizados em tempo real
```

---

## 🧪 Teste Local

```powershell
.\register-bling-webhook.ps1 -Command test
```

Você deve ver: ✓ HTTP 202 - Aceito

---

## 📝 Passos no Bling (Para cada webhook)

1. Vá para: **Configurações → Integrações → Webhooks**
2. Clique em: **+ Novo Webhook**
3. Preencha com os dados acima
4. Clique em: **Salvar**

Pronto! ✅ Repita para cada um dos 4 webhooks.
